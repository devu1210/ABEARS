PROBLEM STATEMENT:
Currently, the entertainment industry faces a "Content Overload" crisis with over 100,000 titles spread across multiple streaming platforms. Users spend nearly 20% of their viewing time just deciding what to watch, leading to decision fatigue. Additionally, traditional rating systems often fail to capture real-time audience sentiment nuances (sarcasm, context), and content producers lack data-driven tools to accurately forecast the popularity of a movie or show before its production or release. Predicting content success and providing relevant, personalized recommendations across fragmented datasets (Netflix, Amazon Prime) has become a major concern for the modern digital ecosystem.

PROPOSED SOLUTION:
The proposed system, WatchWise, addresses these challenges by integrating machine learning and deep learning techniques to provide intelligent entertainment insights and personalized recommendations. The solution consists of the following components:

Data Collection:
- Gather large-scale movie and TV show datasets from Kaggle (specifically Netflix and Amazon Prime titles).
- Include 18,700+ records featuring Title, Genre, Language, Release Year, Plot Overview, and Ratings.

Data Preprocessing:
- Clean and handle missing values in descriptions and genres.
- Feature engineering: Combine text features (Title + Genre + Overview) into a "Combined Metadata" field for vectorization.
- Implement language-based filtering and data normalization for consistent scaling.

Machine Learning Algorithm:
- Recommendation System: TF-IDF vectorization with Cosine Similarity for content-based matching.
- Sentiment Analysis: Deep Learning implementation using a Pre-trained BERT model (HuggingFace) for context-aware review analysis.
- Popularity Prediction: Random Forest Regression model trained on numerical attributes like Budget, Runtime, and Genre Density.

Deployment:
- Develop a responsive web application using Vanilla HTML, CSS (Glassmorphism), and JavaScript.
- Backend APIs developed using FastAPI (Python) to serve real-time model predictions with low latency.

Evaluation:
- Assess model performance using Accuracy, Mean Absolute Error (MAE), and Cosine Similarity Relevance scores.
- Benchmark system response time for real-time interaction.

ALGORITHM & DEPLOYMENT:

Algorithm Selection:
- TF-IDF + Cosine Similarity: Selected for its efficiency in high-dimensional text matching, enabling instant relevant recommendations.
- BERT (Deep Learning): Transformer-based model selected for superior contextual understanding of audience sentiment over traditional VADER/NLTK methods.
- Random Forest: Selected for popularity prediction due to its robustness against outliers and ability to handle non-linear relationships in movie features.

Data Input:
- Input movie titles for similarity search.
- Text-based reviews for sentiment analysis.
- Numerical features (Budget, Runtime, Genres) for popularity scoring.

Training Process:
- TF-IDF matrix generated from the unified 18,700+ title dataset.
- Random Forest model trained using Scikit-Learn with a 70/30 train-test split and hyperparameter tuning.
- BERT model loaded as a pre-trained multilingual transformer for zero-shot or fine-tuned sentiment classification.

Prediction Process:
- User interaction triggers a REST API request via FastAPI.
- The backend processes text via TF-IDF or BERT and numerical data via Random Forest.
- Results are dynamically rendered on the UI with sub-1.5 second latency.
